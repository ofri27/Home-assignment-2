Projects

Gui Project

In the C# course we did a GUI project that included creating a sales system for the topic we choose. In the project I took part in, we implemented a sales system for Student Day at Rupin College. The system included the ticket sales screen + the sale of transportation tickets. We direct manipulation of graphical icons, such as buttons, scroll bars, windows, tabs, menus, cursors, and the mouse pointing device.

"Promotion of girls to the technological world" project

As part of the scholarship that I have been doing for the second year, this year I joined a program to promote girls in youth villages to the worlds of programming and technology. As part of the program, we build an app with the girls through the "App Inventor" system where the girls create an app for a problem that bothers them in the world with an emphasis on "climate and a green world" or "world hunger". The program includes a student and two mentors from the high-tech world who lead the program in the village. As part of the program, we work with the girls on defining what a problem is and choosing a problem that affects their world, defining possible solutions, creating the appropriate app and finally participating in a global competition "Girls+" with two stages: semi-finals, and finals.
